# Soluna

![Banner](https://raw.githubusercontent.com/Soluna-Development/API/refs/heads/main/assets/static/background-with-logo.png)

A Roblox script loader supporting 20+ games with a clean interface, persistent settings, and auto-load functionality.

---

## Getting Started

Execute the following in your preferred executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/Soluna-Development/main/refs/heads/main/src/Loader.lua", true))()
```
